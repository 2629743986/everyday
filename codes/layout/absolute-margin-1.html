核心代码

.wrap{
    position: relative;
    width: 80%;
}
.aside{
    position:absolute;
    top:0;
    left:0;
    width:200px;
}
.main{
    margin-left:210px;
}

问题缺陷

由于侧边栏.aside设置position:absolute，使文档脱离常规流，最终会导致侧边栏.aside无法撑开父元素的高度，它将会溢出父元素区域

假设你左侧的高度是不能固定，这种布局方式不推荐的
